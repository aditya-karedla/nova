## Quick Example

```tsx
import { ThemeProvider, Button, Card } from "@nova/core";
import "@nova/core/styles.css";

function App() {
  return (
    <ThemeProvider>
      <Card>
        <Card.Header>
          <Card.Title>Welcome to Nova</Card.Title>
        </Card.Header>
        <Card.Content>
          <p>Start building amazing applications!</p>
        </Card.Content>
        <Card.Footer>
          <Button>Get Started</Button>
        </Card.Footer>
      </Card>
    </ThemeProvider>
  );
}
```

## Philosophy

Nova is built on three core principles:

1. **Accessibility is non-negotiable** - Every component meets WCAG 2.1 AA standards
2. **Developer experience matters** - TypeScript, great documentation, and intuitive APIs
3. **Flexibility without complexity** - Sensible defaults with powerful customization options

## Tech Stack

- **React 18** - Modern React with hooks and concurrent features
- **TypeScript** - Full type safety and IntelliSense support
- **Tailwind CSS** - Utility-first styling for rapid customization
- **Radix UI** - Accessible, unstyled component primitives
- **Vite** - Lightning-fast build tooling
